# 约束规则统一使用派生的维度等级

所有计分维度无论接收评级还是 0～100 分数，都先形成计算分，再按当期统一评级区间派生维度等级；核心维度与任一维度约束只使用该维度等级。固定规则为：核心维度为 C 时阶段等级强制为 C，核心维度为 B 时阶段等级最高为 B，任一维度为 C 时阶段等级最高为 B。原先按阶段模式分别维护的评级约束配置和分数阈值配置被取消；红线认定强制 C 仍是独立且不可关闭的硬约束。

**Considered Options**：混合计分子表单可以按每个维度的输入类型分别执行评级规则或分数阈值，但管理员仍需维护两套可能漂移的等级边界，且同一计算分可能得到不同约束结论。统一使用当期评级区间派生维度等级，使评级区间成为唯一边界，也让规则能够跨输入类型解释。

**Consequences**：评级区间变化会同时影响阶段综合分映射和维度约束判断，必须按 ADR-0026 对受影响结果统一重算。配置模板不再保存 `WEIGHTED_RATING` 与 `WEIGHTED_SCORE` 两套约束档案，计算明细需要保留每个维度的计算分、派生等级及触发的约束原因。
