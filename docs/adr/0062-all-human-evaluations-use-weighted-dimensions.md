# 三类人工评估统一使用加权计分维度

员工自评、360°评估和上级评估统一使用 ADR-0061 定义的加权计分维度，不再为员工自评保留独立的直接定级表单模型。三类人工子表单都必须至少包含一个计分维度；单个占比必须大于 0%、不超过 100% 且最多两位小数，全部计分维度占比严格合计 100%；每个子表单必须有且只有一个核心计分维度，非计分维度不能成为核心。默认员工自评只包含一个“自评结论”评级计分维度，占比 100% 且为核心维度，因此员工选择的评级仍会形成相同的自评阶段等级；未来增加多个自评计分维度时，则按占比计算自评阶段结果。AI 建议不属于人工表单，仍可直接定级。

**Considered Options**：保留员工自评直接定级改动较小，但会让自评计分维度成为没有占比和核心标记的例外，管理员仍需理解两套表单规则。把默认自评表达成占比 100% 的单个评级维度不改变现有填写结果，同时统一三类人工表单的配置、校验和计算边界。

**Consequences**：员工自评的独立总体等级输入需要迁移为维度计分输入；三类人工表单的计分维度占比均合计 100%，并通过同一计算入口生成阶段等级。默认数据、提交完整性校验、结果展示和历史快照需要保留原自评等级的业务含义。
