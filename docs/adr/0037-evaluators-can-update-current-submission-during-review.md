# 校准完成前评估人可以更新并重新提交当前答卷

系统不提供“退回评估”功能。员工自评、360°评估和上级评估都由原评估人直接修改本人当前答卷并重新提交，直到该参与者首次形成正式校准结果并按 ADR-0039 锁定。重新提交仍须通过完整性校验，并立即按当前周期配置重新计算相关个人结果、关系汇总和阶段结果。每名评估人在同一参与者、同一评估类型下只保留一份当前答卷；修改直接更新当前评估项结果，不创建或保留答卷历史版本。

**Considered Options**：提交后锁定并由管理者退回可以形成严格审批式审计链，但会增加退回操作和修订版本管理，与业务希望评审阶段内可持续完善评价的方式不符。让原评估人在开放阶段内直接更新当前答卷，操作更直接，也避免把绩效填写设计成审批流程。

**Consequences**：数据模型不需要评估提交修订版本、来源版本或退回状态；唯一约束应保证每名填写人只有一份当前答卷。提交状态表示“当前内容已完成完整性校验”，重新提交时覆盖当前结果并触发重算。ADR-0036 中正式提交不可变与退回修订的决定失效，但其完整性校验仍然有效。阶段结果引用当前答卷而非答卷版本；写接口必须检查参与者是否已经评估锁定。
