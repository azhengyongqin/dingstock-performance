# 申诉每人每周期限一次

参与者状态机曾允许 `RE_CONFIRMING → APPEALING` 循环，即员工对申诉处理结果可以无限再申诉。结合「存在未闭环申诉时周期不可归档」的规则，一个坚持申诉的员工可以让整个周期无限期无法归档。我们决定：**系统内申诉每人每周期限一次**——删除 `RE_CONFIRMING → APPEALING` 状态边；再次确认只有两个出口：确认（含超时视同确认）或转「HR 介入」线下通道（系统只记录处理意见与面谈，不回到申诉态）。

**理由**：一次正式申诉 + 面谈 + 改判/维持已构成完整程序正义，之后的分歧本质是劳动关系问题，不该由状态机建模；同时保证归档阻塞有限步收敛（申诉窗口截止 + 再确认超时视同确认 + 无二次申诉）。

**Consequences**：归档流程数学上必然排空，无死锁风险；若未来需要二次申诉，需同时重新设计归档阻塞规则，两者是一对约束。
