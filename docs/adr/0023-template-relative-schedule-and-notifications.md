# 配置模板包含相对日程与通知预设

> 补充：员工自评、360°和上级评估的截止时间按 ADR-0041 仅触发通知；开始时间按 ADR-0051 作为任务开放门槛。

配置模板除评估规则、评审员关系权重和评估表单模板版本外，还保存各评估任务的开始时间偏移、阶段重叠策略、填写提醒截止时间、提醒频率、通知接收人与抄送规则。模板不保存任何绝对日期；创建绩效周期时，系统根据 HR 填写的计划启动时间生成实际日程，HR 可以调整生成结果。填写提醒截止时间不是关闭任务的硬截止。日程与通知预设随配置模板版本发布，并复制为周期配置快照。

**Considered Options**：每个周期完全手填时间和提醒最灵活，却重复劳动且容易遗漏；复制上一个周期的绝对时间会产生明显错误。用相对时长和通知策略生成可编辑的实际窗口，既能复用公司流程，又保留周期级日历调整能力。

**Consequences**：配置模板版本需要结构化日程与通知预设；周期创建流程需要计划启动时间、日程预览和冲突校验。生成后的绝对窗口属于周期快照，不随模板更新；旧模板套用 PRD 中“时间窗口和通知规则不进入模板”的范围被本 ADR 取代。
