# 当前版本固定 S/A/B/C 四档评级

当前公司绩效评级固定为 S、A、B、C 四档，顺序为 S > A > B > C。管理员不能新增、删除评级档位或修改评级符号，但可以在配置模板中调整评级名称、说明、连续覆盖 0～100 的分数区间、评级映射分以及评语必填条件。红线强制 C、核心维度封顶 B 等约束可以直接引用稳定的评级语义。

**Considered Options**：支持任意评级数量与符号能服务更多制度，但所有等级比较、强制最低档、封顶档位、颜色、报表与历史趋势都必须改为动态解释，且与当前明确的 SABC 制度不符。固定档位和顺序，同时开放区间与映射参数，已经覆盖现阶段需要的可配置范围。

**Consequences**：模板和周期规则校验必须确保 S/A/B/C 各出现一次、顺序稳定、区间连续覆盖 0～100，不能再接受任意评级集合。评级仍可按 ADR-0005 保存在评估规则 JSON 快照中，但 API 与前端编辑器只能修改允许变化的属性。
