# 评估填写截止时间目前只触发飞书通知

员工自评、360°评估和上级评估的填写截止时间均为软截止时间，目前只用于触发飞书提醒。时间到达后，任务不关闭、状态不自动变化，评估人仍可首次提交、修改和重新提交。系统不再因截止时间自动产生自评弃权或 360°逾期未评，也不在截止时固化 360°结果。评估写入权限最终在单个参与者形成首次校准结果时，按 ADR-0039 统一锁定。

**Considered Options**：把截止时间作为硬截止可以自动收口流程，但会阻止迟交评价进入校准，与当前业务更重视收集完整输入的方式不符。将截止时间只用于通知，可以保留管理提醒，同时让评估人在校准前持续补充内容。

**Consequences**：定时任务只能发送飞书通知和记录通知结果，不能修改评估任务或参与者状态。前后端不得因当前时间超过截止点而禁用保存、提交或重新提交。原有自评弃权和逾期未评状态、截止固化逻辑应删除或停止使用；360°当前汇总随正式提交持续重算，最终以参与者评估锁定时的有效提交集合为准。
