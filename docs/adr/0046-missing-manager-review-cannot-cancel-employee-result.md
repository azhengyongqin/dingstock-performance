# 上级评估缺失不能取消员工本周期结果

员工已经完成自评但上级评估始终未提交时，不能仅因 Leader 未履职就把员工标记为“本周期无绩效结果”。该员工不能进入校准，并持续阻塞周期归档；HR/Admin 必须催办，必要时由 HR 更换考核 Leader，再由新的责任 Leader完成上级评估。HR/Admin 不得代替 Leader填写。只有员工本身满足离职、组织调整等中途退出条件时，才能按中途退出收口，Leader 未提交本身不是退出理由。

**Considered Options**：允许用无绩效结果快速收口可以避免周期阻塞，但会把管理者未完成任务的后果转嫁给员工；允许 HR/Admin 代评则混淆组织授权与实际管理评价责任。通过催办或更换责任 Leader完成上级评估，能保证员工拥有结果且评价主体明确。

**Consequences**：无绩效结果接口必须验证员工自评确实缺失，不能接受“上级评估缺失”作为原因。工作台和通知需要对未完成上级评估持续升级提醒，并提供更换考核 Leader入口。周期归档校验必须列出仍缺少上级评估的参与者，不能静默跳过。
