# 员工自评和上级评估是校准硬性前置条件

单个参与者形成首次校准结果前，员工自评和上级评估都必须已经正式提交且当前有效。上级评估提供校准前权威等级；员工自评虽然只作为参考，也属于必须完成的正式流程输入。360°评估允许部分提交或完全无数据，不阻塞校准；有提交时按当前有效提交集合计算，完全无提交时形成无数据阶段结果。填写提醒截止时间只是通知触发点，不会把缺失自评转为弃权，也不会豁免这两个前置条件。

**Considered Options**：只要求上级评估可以保证系统拥有权威等级，但会让员工在完全未参与自评的情况下直接进入校准，不符合本流程对员工参与的要求。要求所有 360°也完成则会让参考性评价无限阻塞流程。将员工自评与上级评估设为硬前置、360°设为可选参考，明确了必要输入和补充输入的边界。

**Consequences**：校准接口和工作台必须检查当前有效员工自评与上级评估是否存在且完整，任一缺失时禁用或拒绝校准并展示待办原因。始终未提交必需自评的参与者按 ADR-0044 以本周期无绩效结果收口；原有自评弃权后继续校准的规则不再适用。
