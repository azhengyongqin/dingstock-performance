# 360°评审员不能选择无法评价

360°评审任务不提供“了解不足、无法评价”或类似主动收口动作。评审员答卷只保留草稿与正式提交两类状态；只有完整正式提交才产生分数并进入关系平均，未提交草稿不产生分数。填写提醒截止时间仍只触发通知，参与者完成校准后未提交草稿保持草稿并锁定。

**Considered Options**：允许评审员主动声明无法评价可以减少被迫猜测和后续催办，但会新增任务终态，并可能让评审员以此规避已接受的评价责任。当前业务明确不允许该操作，因此继续使用草稿或正式提交的简单状态模型。

**Consequences**：前端不展示跳过、拒绝或无法评价按钮，后端也不接受对应状态。360°汇总继续只读取成功提交内容；未提交者是否进入最终计算由参与者校准锁定时的有效提交集合决定。
