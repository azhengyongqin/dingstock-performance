# 受结构变化影响的答卷必须重新提交

周期退回草稿并修改表单结构后，如果变化影响答卷完整性或计算语义，例如新增必填项、修改计分项、改变输入类型或维度归属，所有受影响的已提交答卷统一转为草稿，必须由原评估人重新提交。仍能按稳定评估项标识与兼容类型匹配的旧值自动预填；无法匹配的旧值保留在历史配置数据中但不进入新答卷与计算。重新提交前，旧答卷不参与新配置计算；员工自评和上级评估继续阻塞校准，360°按草稿排除。只有标题、说明、占位提示等非结构文案变化时，答卷保持已提交且无需重新提交。

**Considered Options**：自动把旧答卷视为新结构下已提交可以减少操作，但新增必填项或改变计分语义后，旧答卷并未满足新的提交承诺。全部清空重填又浪费仍兼容的数据。转为草稿并预填兼容值，能让填写人明确确认新结构，同时减少重复填写。

**Consequences**：结构变更影响分析必须精确定位受影响的表单、参与者和填写人；重新激活前展示需要重新提交的人数与项目。任务与通知系统需要重新产生待办，计算服务排除这些草稿。非结构文案变更必须有独立分类，不能误触发大规模重新提交。
