# 进行中周期允许修改计算配置并统一重算

管理员可以在周期归档前修改评级区间、映射分、维度占比、核心维度、约束规则和 360°关系权重等计算相关配置。如果已经存在受影响的评估提交，系统必须先展示影响范围与阶段结果变化预览，要求填写原因并二次确认；确认后生成新的周期配置版本，基于原始填写值对全部受影响提交和阶段结果统一重算。旧配置版本、旧计算明细和旧阶段结果永久保留，不允许同一阶段长期混用不同计算版本。按 ADR-0027，重算不自动改写已经形成的校准结果和员工最终结果。维度或表单字段结构变化不属于本 ADR 的在线重算范围，按 ADR-0057 处理。

**Considered Options**：有提交后彻底锁定配置最安全，但无法纠正真实的模板或规则错误；允许修改但不重算会让权重、区间与结果口径永久不一致；只重算后续提交会让同一周期内部不可比较。可修改、全量重算并保留版本链，既允许纠错，也维持结果一致与审计可追溯。

**Consequences**：周期配置需要版本标识，评估项结果和阶段结果必须关联所用配置版本，并支持从不可变原始填写值重放计算。配置编辑接口需要影响分析、预览、必填原因、确认和原子重算能力；重算失败必须整体回滚。影响预览必须分别显示阶段结果变化与因已有校准而保持不变的最终结果。旧的“进行中修改计算配置但不回溯重算”设计被本 ADR 取代，已归档周期仍永久不可修改。
