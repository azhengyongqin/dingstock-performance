# 表单模板使用受控评估项组件库

评估表单模板不提供任意 Schema、HTML 或脚本编辑能力，而是从受控评估项组件库组合表单。首期类型固定为评级选择、0～100 分数输入、单行文本、多行文本、Markdown、单选、多选、文件附件和链接。每个评估项统一配置标题、说明、占位提示、是否必填、字数或数量限制及排序；只有评级选择和分数输入能够参与绩效计算，其他类型只保存结构化内容。

**Considered Options**：开放任意 JSON Schema 或自定义组件能获得最大灵活性，但会把渲染安全、数据校验、移动端兼容、导出格式和历史回放都变成不可预测的插件问题。受控组件库已经覆盖现行自评、360°、上级和晋升表单，同时允许系统为每种类型提供稳定的数据契约。

**Consequences**：模板与周期快照需要显式评估项类型和结构化配置，前端按类型注册标准编辑器与只读渲染器，后端按相同类型执行提交校验。新增评估项类型属于受控产品演进，需要同步扩展模板编辑、填写、结果展示、导出和兼容测试，不能只在前端临时增加组件。
