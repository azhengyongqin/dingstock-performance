# 阶段重算不自动改写已有校准结果

配置修改触发的重算只更新员工自评、360°评估、上级评估及其阶段结果。一旦参与者已经形成由 Leader、HR 或授权管理者作出的校准结果，该人工结论就成为阶段计算与员工最终结果之间的权威边界；后续阶段重算不得自动改写校准结果、已推送结果或员工确认。新的阶段结果可作为人工复核信息，如确需改变最终结果，必须显式重新校准并追加一条新记录，旧校准记录永久保留。红线认定的强制 C 属于独立硬约束，不视为普通配置重算。

**Considered Options**：让配置重算自动穿透校准并重新打开员工确认，可以保持最终结果与最新算法完全一致，但会推翻已经完成的人工管理决策，使最终结果失去稳定边界。完全禁止校准后重算又会阻止修正评估阶段数据。阶段结果可重算、校准结果不自动变化，同时允许显式重新校准，能兼顾纠错与最终结果稳定性。

**Consequences**：阶段结果与校准结果必须独立版本化；重算影响预览需要明确区分“阶段结果变化”和“最终结果不变”。员工确认不因普通配置重算自动失效；只有追加的新校准结果真正改变员工可见最终结果时，才进入重新推送与确认流程。
