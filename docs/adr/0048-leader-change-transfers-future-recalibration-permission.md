# 校准后更换 Leader 只转移未来重新校准权限

参与者已经形成校准结果后再更换考核 Leader，新 Leader立即获得该参与者敏感评估明细的查看权和追加重新校准权限，旧 Leader立即失去这些权限。既有校准结果继续有效，不因 Leader变化自动改变；上级评估答卷仍受评估锁定保护，任何人都不能再修改。新 Leader如需调整结果，只能追加重新校准记录。历史评估与校准记录永久保留当时的提交人或操作人，不随责任人变化改写。重新校准导致等级变化时，继续按 ADR-0031 生成新结果版本并要求员工重新确认。HR/Admin 的既有授权不受 Leader变化影响。

**Considered Options**：让旧 Leader继续负责本周期可以保持原管理关系，但会让已经不再承担责任的人持续访问敏感数据；更换 Leader时自动重做评估或校准又会破坏锁定与人工决定边界。只转移未来查看和重新校准权限，可以保持历史真实性并让当前责任人处理后续事项。

**Consequences**：对象级授权必须读取参与者当前考核 Leader快照，而不能只读取历史提交人或校准人。更换操作需要立即使旧权限失效并授予新权限；审计、结果版本和校准记录仍按事件发生时的操作者展示。
