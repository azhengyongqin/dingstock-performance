# 本周期无绩效结果只能由 HR 或 Admin 设置

只有授权组织范围内的 HR 和全局 Admin 可以把缺少必需自评的参与者标记为“本周期无绩效结果”。直属 Leader 可以发起申请或提醒，但不能直接终止员工本周期结果。设置时必须填写原因并明确确认该员工全部未完成评估任务将被关闭。周期归档前，HR/Admin 可以填写撤销原因后恢复参评：员工自评、360°和上级评估重新开放，设置前已经成功提交的当前内容继续保留。周期归档后该终态永久不可撤销。

**Considered Options**：允许直属 Leader 直接设置可以减少 HR 操作，但可能让管理者单方面取消员工绩效结果；完全不可撤销又难以纠正误操作或员工恢复参与的情况。由 HR/Admin 控制并允许归档前带原因撤销，兼顾权限边界与可纠错性。

**Consequences**：设置与撤销接口需要对象级组织权限、必填原因、二次确认和审计记录。状态切换需要原子地关闭或恢复该参与者的待办和评估写入权限，但不删除已有答卷，也不创建答卷历史版本。归档校验把该终态视为已收口参与者。
