# 已归档周期不能退回

`ARCHIVED` 是绩效周期永久不可逆终态，任何角色包括超级管理员都不能将其退回 `ACTIVE`、`SCHEDULED` 或 `DRAFT`。超级管理员按 ADR-0052 拥有的整体退回权限只适用于 `ACTIVE` 周期。已归档结果属于正式历史绩效档案，不允许重新开放答卷、失效确认或改写等级。归档后发现重大错误时，只能创建独立的归档结果更正记录或补充周期，保留原结果和更正依据。

**Considered Options**：允许超级管理员退回归档周期能提供最大纠错能力，但会使已经进入员工档案、报表和下游使用场景的正式结果失去稳定性。使用独立更正记录表达事后纠错，可以同时保留原始事实与后续处理。

**Consequences**：周期状态机不提供任何 `ARCHIVED` 出边，所有写接口必须拒绝归档周期。未来实现归档后纠错时必须使用独立追加模型，不能复用周期退回或普通重新校准接口。
