# 校准锁定时 360°草稿保持草稿状态

参与者形成首次校准结果并评估锁定时，不为尚未正式提交的 360°答卷新增“校准时未提交”或其他关闭状态。答卷保持草稿状态，但因参与者已锁定而不能继续修改或提交。从未成功提交过的草稿不产生分数、不进入关系平均；如果评审员曾经成功提交，当前只是保存了尚未重新提交的更新草稿，则继续以上一次成功提交内容参与计算。只有全部 360°评审员都不存在有效成功提交时，360°阶段结果才固化为无数据；部分人员有有效提交时，按这些提交及有效关系归一化计算。

**Considered Options**：把未提交指派转换成专门终态可以更直观地区分流程收口，但会增加状态，并与业务希望直接保留草稿的表达不一致。删除草稿又会丢失填写人尚未提交的当前内容。保留草稿状态、由参与者评估锁定控制可写性，可以减少状态转换，同时保持有效提交与未提交编辑内容的边界。

**Consequences**：答卷状态本身不能单独决定是否可编辑，写接口还必须检查参与者评估锁定。360°计算查询只读取最后一次成功提交内容，不读取当前未提交草稿。页面需要将“草稿但参与者已校准锁定”展示为只读状态，并明确其未进入最终 360°汇总。
