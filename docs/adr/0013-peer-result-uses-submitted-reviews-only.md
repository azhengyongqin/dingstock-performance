# 360°正式结果只汇总已提交评价（截止行为已被 ADR-0041 取代）

> 状态：只汇总正式提交、无提交时产生无数据结果的决定继续有效；截止时转“逾期未评”并固化结果的决定已被 ADR-0041 取代。

评审员关系是否有效以是否至少存在一份正式提交的 360°评价为准，而不是以是否曾经指派评审员为准。同一关系内只平均已提交评价；已指派但未提交的人不产生分数。如果参与者评估锁定时仍没有任何正式提交，360°阶段形成无数据结果，分数、初始等级和阶段等级均为空，并继续后续流程。填写提醒截止时间不再终结任务或固化结果。

**Considered Options**：要求所有已指派评审员提交才能推进，能保证计划样本完整，但任何失联或逾期评审员都可能无限阻塞参与者；把未提交或完全无数据视为零分、C 或默认 B 会伪造评价并系统性影响被评估人。按正式提交结果计算、把未提交任务转为独立终态，并显式记录无数据结果，可以保留任务责任记录而不污染绩效分数。

**Consequences**：阶段结果需要支持无数据状态与可空分数/等级；评审员指派不再需要逾期未评终态。计算明细必须记录实际参与的提交和被排除的未提交指派。当前“待提交数量必须为零才能推进”的代码需要改为允许校准人员在规则允许时以当前已提交评价形成参考结果。
