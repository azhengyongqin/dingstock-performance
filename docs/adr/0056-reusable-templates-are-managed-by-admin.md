# 复用模板默认只由 Admin 管理

配置模板和评估表单模板的创建、草稿编辑、版本发布与归档默认只开放给 Admin。Admin也负责调整等级规则、360°关系权重等进入复用模板的全局配置。HR可以查看并选用已发布模板创建周期，也可以在授权周期内修改周期配置快照，但不能反向修改、发布或归档复用模板。Leader、员工和 360°评审员没有模板管理权限。模板发布、归档以及从已发布版本创建新草稿都记录审计日志。未来如需授权部分 HR管理模板，应新增独立模板管理员权限，不默认扩展全部 HR角色。

**Considered Options**：默认允许所有 HR发布模板可以提高分散运营效率，但模板会跨周期复用并决定计算规则，错误影响面较大。只允许 Admin管理复用资产、允许 HR在具体周期内调整快照，能把全局标准与周期运营权限分开。

**Consequences**：模板接口使用 Admin权限守卫，周期快照编辑使用 HR组织范围或 Admin权限。前端需要为 HR提供只读模板详情与版本选择，不展示发布、归档等操作。模板管理动作全部进入审计日志。
