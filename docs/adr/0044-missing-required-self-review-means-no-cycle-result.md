# 始终缺少必需自评的员工本周期无绩效结果

员工自评是首次校准的硬性前置条件。参与者始终没有正式提交自评时，不允许 Leader、HR 或 Admin 代填，也不使用默认等级继续校准；该员工以“本周期无绩效结果”终态收口。该终态不等同于中途退出：员工仍在本周期参评名单中，但因缺少必需输入而不生成校准结果、最终等级、结果推送或员工确认，并且不再阻塞周期归档。

**Considered Options**：持续等待会让单个未自评员工无限阻塞周期，允许他人代填则破坏员工自评的责任与真实性，跳过自评继续定级又与 ADR-0042 的硬前置条件冲突。显式记录本周期无绩效结果，能如实表达流程未完成且让周期正常收口。

**Consequences**：参与者状态机需要独立的无绩效结果终态和原因，报表必须将其与中途退出、无 360°数据、正常绩效等级分别统计。无结果参与者不进入等级分布和绩效档案结果，但保留本周期参与记录与缺失原因。具体设置与撤销权限按 ADR-0045 执行。
