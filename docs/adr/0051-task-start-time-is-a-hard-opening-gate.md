# 任务开始时间是硬开放门槛

员工自评、360°评估和上级评估可以配置不同开始时间并允许重叠。开始时间到达前，填写人只能看到任务预告，不能保存草稿、首次提交或重新提交；到达后任务自动开放。360°和上级评估开放不强制等待员工自评提交，自评缺失时只显示暂无自评参考，后续自评提交后自动可见。任务一旦开放，不因填写提醒截止时间关闭，持续可填写到该参与者完成校准、被标记为本周期无绩效结果、中途退出或周期归档。HR/Admin 可以带原因调整尚未到达的开始时间或手动提前开放，已经开放的任务不能重新关闭。

**Considered Options**：让开始时间也只作为通知点会导致任务在周期启动后全部立即可填，削弱日程编排意义；要求 360°或上级评估等待自评完成又会让一个必需输入串行阻塞其他评价。硬开始门槛、软截止提醒和非依赖式并行开放，既保留节奏控制，也允许流程高效推进。

**Consequences**：任务可写判断必须同时检查周期是否进行中、任务是否已开放和参与者是否已收口锁定，不能用截止时间拒绝写入。开始时间触发任务开放和飞书通知；调整或提前开放必须审计。已开放标识需要独立保存或从不可逆事件推导，防止后续修改日程把任务重新关闭。
