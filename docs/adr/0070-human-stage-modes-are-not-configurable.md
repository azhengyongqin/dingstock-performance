# 三类人工评估不再配置阶段模式

员工自评、360°评估和上级评估固定使用统一的维度加权计算，评级或 0～100 分数由每个计分维度单独定义，同一子表单可以混用；因此配置模板和周期配置不再保存或展示 SELF、PEER、MANAGER 阶段模式。AI 建议可以直接产生等级，但该行为属于内部能力，不作为管理员可配置的表单阶段模式。

**Considered Options**：保留阶段模式字段便于兼容旧结构，但它会与维度计分方式重复，并重新引入“同一子表单只能评级或只能评分”的冲突。阶段类型本身已经足以确定计算入口，计分细节由表单维度完整表达。

**Consequences**：配置模板、周期配置版本、API、管理页面、发布校验、计算预览和阶段结果不再依赖阶段模式枚举。配置模板只维护评级区间、评级映射分、统一等级约束、360°关系占比、日程与通知等规则；旧阶段模式字段和两套约束档案在开发数据重置中直接移除。
